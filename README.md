# between
Small utility which extracts data between some prefix and suffix

# Usage
```
USAGE:
    between [OPTIONS] --prefix <PREFIX> --suffix <SUFFIX> [INPUT]

ARGS:
    <INPUT>    Path to file to read data from

OPTIONS:
    -h, --help               Print help information
        --keep-prefix        Keep prefix in the output (default: no)
        --keep-suffix        Keep suffix in the output (default: no)
        --prefix <PREFIX>    Prefix to search for
    -s, --stdin              Read data from standard input
        --suffix <SUFFIX>    Suffix to search for
    -V, --version            Print version information```
